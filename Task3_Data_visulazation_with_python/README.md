 IPL 2026 Data Analysis (EDA Project)

 #Overview

- Exploratory Data Analysis (EDA) on IPL dataset
- Focus on team & player performance
- Extract insights using data visualization

---

 #Objectives

- Analyze total runs by teams
- Identify top-performing players
- Understand match trends
- Create meaningful visualizations

---

 #Dataset

- IPL dataset (CSV format)
- Includes:
  - Teams
  - Players
  - Runs
  - Match details

---
#TechStack

- #Python 
- #Pandas
- #NumPy
- #Matplotlib
- #Seaborn
- #JupyterNotebook

---

 #Workflow

- #DataLoading → Import dataset
- #DataCleaning → Handle missing values & duplicates
- #EDA → Grouping & aggregation
- #Visualization → Charts & graphs

---

 #KeyInsights

- Some teams consistently score higher
- Top players dominate run scoring
- Batting plays major role in results
- Performance varies across matches

---

 #Visualizations

- Bar charts 
- Line graphs 
- Comparative analysis

---

 #Conclusion

- Clear understanding of team performance
- Identified top players
- Discovered match trends
